01.1 · propósito

Propósito y alcance del documento

Esta guía define la identidad visual y verbal de ForMeta. Existe para que cualquier persona que trabaje con la marca — interna o externamente — tome decisiones coherentes sin tener que preguntar.

No es un manual de restricciones. Es una descripción de quiénes somos, traducida a decisiones concretas de diseño, lenguaje y comunicación.

El documento cubre: sistema de identidad visual (logo, color, tipografía, símbolo), voz y tono, y criterios de aplicación en los contextos donde ForMeta aparece. No cubre decisiones de producto, arquitectura técnica ni procesos internos.

01.2 · cómo usar

Cómo usar esta guía

Las secciones marcadas como fundacional (marca, logo, color, tipografía) no se negocian — son la base. Las secciones marcadas como aplicación (componentes UI, papelería, plantillas) son puntos de partida que se adaptan al contexto.

Cuando algo no esté cubierto aquí, la pregunta correcta es: ¿esto es coherente con el manifiesto de la sección 02? Si la respuesta es no, no se hace. Si la respuesta es sí, se hace y se documenta aquí para la próxima versión.

v1.0 incluye los 13 capítulos completos del sistema. Los únicos elementos pendientes para v2.0 son fotografía editorial, calibración Pantone/CMYK, licencia comercial de Canela y componentes adicionales (data table, calendar picker, command palette). Ver el changelog del cap. 13.
